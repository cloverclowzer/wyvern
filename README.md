# Wyvern Chat

A polished chatroom built with Express and a JSON-backed database.

## Features
- Register/login system
- Persistent JSON database
- Default Admin account: `Admin` / `whatthesigma`
- Mod powers for admins:
  - Delete messages
  - Ban users
  - Grant blue and gold verification badges
  - Allow users to hold both badges at once
- Discord-style channels

## Run locally
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the server:
   ```bash
   npm start
   ```
3. Open `http://localhost:3000`
